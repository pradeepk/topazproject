Mulgara resolver to implement string-comparisons models.

Example: 

      create <local:///topazproject#str> <http://topazproject.org/models#StringCompare>;
      select $s $p $o from <local:///topazproject#foo>
        where $s $p $o
        and $o <topaz:equalsIgnoreCase> 'Jane Bar' in <local:///topazproject#str>;

Currently supported operations are:

topaz:equalsIgnoreCase
case-insensitive equality comparison
topaz:lt
less than
topaz:le
less-or-equal than
topaz:gt
greater than
topaz:ge
greater-or-equal than
(where 'topaz' is an alias for 'http://rdf.topazproject.org/RDF/'). Comparisons can be made on literals or URI's.

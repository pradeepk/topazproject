Exposes fedoragsearch's operatons API.

To get a handle to the API, call new OperationsServiceLocator().getFgsOperations(URL) with the URL of the running fedoragsearch webservice.

Most of the API parameters provide reasonable default values if null is passed as the parameter. For example, the Operations.gfindObjects() is usually configured to return all its results as non-transformed XML if null is passed as the resultPageXslt (last paramter).

See also documentation on trac wiki: FedoraSearch
